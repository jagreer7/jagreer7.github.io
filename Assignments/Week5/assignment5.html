From the website duda, I learned how to use the "abbr" in line tag. This tag can be applied to strings and assign them an abbreviation. Then when the user hovers over them, they see the full title as defined in the tag. For example, the table of contents header is abbreviated. I could write out any phrase, and then using the abbr tag, "Hello Wonderful World" becomes HWW!
