My Works


Text

"Text" is a class I took in the Fall of 2022. This class focuses on typography both from a historical but also a fundamental understanding perspective. This class gave me a rich understanding of the importance of typography in the world of design.


Check out my Text projects!

Object

"Object" is a class I took in the Fall of 2022. This class focused on basic and complex electronics, from simple breadboards to Arduino. We used electronics as well as users' inputs and outputs to create engaging and meaningful projects from an interactive box to a cooperative game.


Check out my Object projects!

Form

"Form" is a class I took in the Fall of 2022. This class taught 3D Modelling and fabrication through Autodesk's Fusion 360 and Ultimaker's CURA. In this class, I learned modeling and fabrication techniques and applied them to solve exciting challenges.


Check out my Form projects!

Web

"Web" is a class I took in the Spring of 2022. This class is an introduction to the coding languages HTML, CSS, and Javascript. This very website you are viewing was the midterm project for this web class. In this class, I learned from the ground up the fundamentals of these coding languages and how to use them to create my very own website. Please enjoy walking through the visual progress of my coding journey.


Check out my Web projects!

Image

"Image" is a class I took in the Spring of 2022. This class is an introduction to the world of images. However, this class wasn't just about snapping photos. This class covered photography, photo editing, Photoshop, and even basic animation. We even covered some of the history and deeper side of photography and images. The projects and challenges this class presented me were "out of this world" fun and I am so excited to share them with you!


Check out my Image projects!

Design Foundations

"Design Foundations" is a class I took in the Fall of 2021. This class served as a way for students to learn about the fundamentals of design. What makes a design good? What makes a design not good? Ultimately, for me, the class was a way for me to use those fundamentals to start working on wonderful designs. Funny enough, the links below are to project posts on an alternate website that was created as a part of the Design Foundations course. Not as cool as this site (which I coded by hand), but an interesting look into the design foundations past.
